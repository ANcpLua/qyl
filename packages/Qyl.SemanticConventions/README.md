# Qyl.SemanticConventions

qyl-owned OpenTelemetry semantic convention attribute keys.

Generated by the Weaver pipeline from the YAML registry under
`eng/semconv/qyl/model/`. Consume `Qyl.SemanticConventions.Attributes.Qyl.QylAttributes`
for any `qyl.*` attribute key.

Schema URL: `https://schemas.qyl.io/1.0.0`

## Usage

```csharp
using Qyl.SemanticConventions.Attributes.Qyl;

activity.SetTag(QylAttributes.IssueId, issueId);
activity.SetTag(QylAttributes.IssueSeverity, QylAttributes.IssueSeverityValues.High);
```
