# docs/done/

Completed or superseded documents. Moved here to keep active docs clean.

## What goes here

- Completed plans
- Superseded roadmap notes
- Temporary execution docs that are no longer active

## What NEVER moves here

- Prompts (`docs/prompts/`)
- Slice `spec.md` and `requirements.md`
- Decisions and references
- Screenshots
- Canonical guidance docs

## Rule

Move the original file, update links, do not leave duplicate shadow copies.
