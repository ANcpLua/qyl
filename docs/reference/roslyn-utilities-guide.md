## ANcpLua.Roslyn.Utilities — Implementation Guide for qyl

This package provides the generator/analyzer infrastructure that qyl's instrumentation generators depend on.
Three layers: generator infrastructure, symbol analysis, runtime.

---

### Layer 1: Generator Infrastructure

#### EquatableArray<T> — Value-equatable collections for incremental pipelines

All generator models MUST use `EquatableArray<T>` instead of arrays/lists for collection properties.
This enables Roslyn's incremental caching (if the model didn't change, skip regeneration).

**Defining models:**
```csharp
// Every collection field in a pipeline record is EquatableArray
internal sealed record TracedCallSite(
    string SortKey,
    EquatableArray<string> ParameterTypes,    // not string[]
    EquatableArray<string> ParameterNames,    // not List<string>
    EquatableArray<TracedTagParameter> Tags); // works with custom records too
```

**Creating from analyzers (extract phase):**
```csharp
// Pattern: .Select().ToArray().ToEquatableArray()
var parameterTypes = method.Parameters
    .Select(static p => p.Type.ToDisplayString())
    .ToArray()
    .ToEquatableArray();

// Empty check — use IsDefaultOrEmpty, not Length == 0
if (input.Left.IsDefaultOrEmpty) return FileWithName.Empty;

// Conditional creation
agents.Count is 0 ? default : agents.ToArray().ToEquatableArray()
```

**Consuming in emitters:**
```csharp
// Length property, indexer access
for (var i = 0; i < parameterTypes.Length; i++)
{
    var typeName = parameterTypes[i];
}

// AsImmutableArray() for interop
typeName.ToGlobalTypeName(typeParamNames.AsImmutableArray());

// String joining
string.Join(", ", parameterNames)  // implicit IEnumerable
```

#### Pipeline Extensions

```csharp
// CollectAsEquatableArray() — replaces .Collect() in incremental pipelines
var meters = context.SyntaxProvider
    .ForAttributeWithMetadataName(
        MeterAnalyzer.MeterAttributeName,
        MeterAnalyzer.CouldBeMeterClass,
        MeterAnalyzer.ExtractMeterDefinition)
    .CollectAsEquatableArray();

// Combine for multi-source pipelines
var combined = tracedCallSites.CollectAsEquatableArray()
    .Combine(agentCallSites.CollectAsEquatableArray());
```

#### IndentedStringBuilder — Auto-indented code generation

**qyl emitters currently use raw StringBuilder with manual spaces. Use this instead.**

`IndentedStringBuilder` tracks indent depth automatically. `BeginBlock()` returns
a disposable `IndentScope` — use with `using` for automatic `}` closure.

```csharp
var sb = new IndentedStringBuilder();
sb.AppendLineNoIndent("// <auto-generated/>");
sb.AppendLineNoIndent(GeneratedCodeHelpers.NullableEnable);
sb.AppendLine();

using (sb.BeginNamespace("Qyl.Generated"))
{
    sb.AppendLine(GeneratedCodeHelpers.ExcludeFromCodeCoverage);
    using (sb.BeginClass("internal static partial", "MyInterceptors"))
    {
        using (sb.BeginMethod("public static", "void", "Intercept", "this MyType @this"))
        {
            sb.AppendLine("// method body — auto-indented to 3 levels");
            using (sb.BeginIf("@this.IsEnabled"))
            {
                sb.AppendLine("@this.Record();");
            }
        }
    }
}

return sb.ToString();
```

**Output:**
```csharp
// <auto-generated/>
#nullable enable

namespace Qyl.Generated
{
    [System.Diagnostics.CodeAnalysis.ExcludeFromCodeCoverage]
    internal static partial class MyInterceptors
    {
        public static void Intercept(this MyType @this)
        {
            // method body — auto-indented to 3 levels
            if (@this.IsEnabled)
            {
                @this.Record();
            }
        }
    }
}
```

**Available block helpers:**
| Method | Opens |
|--------|-------|
| `BeginBlock(opener)` | Generic `{` block |
| `BeginNamespace(name)` | `namespace X {` |
| `BeginClass(mods, name, base?)` | `mods class X : Base {` |
| `BeginMethod(mods, ret, name, params)` | `mods ret Name(params) {` |
| `BeginIf(condition)` | `if (condition) {` |
| `BeginElse()` | `else {` |
| `BeginForEach(type, var, collection)` | `foreach (type var in collection) {` |

**Key details:**
- `AppendLine(text)` auto-prepends current indent. `AppendLineNoIndent(text)` skips it (for `#pragma`, `#nullable`).
- `IndentScope` is a `readonly struct` — zero allocation.
- `Clear()` resets for reuse.

### Layer 2: Symbol Analysis (analyzers only)

#### InvocationMatcher / Invoke DSL — Declarative call-site detection

```csharp
// Simple method matching (DbCallSiteAnalyzer)
private static readonly (...) Matchers = [
    ("ExecuteReader",      Invoke.Method("ExecuteReader"),      DbCommandMethod.ExecuteReader,  false),
    ("ExecuteReaderAsync", Invoke.Method("ExecuteReaderAsync"), DbCommandMethod.ExecuteReader,  true),
];

// With type filtering (GenAiCallSiteAnalyzer)
var matcher = Invoke.Method(methodName)
    .Where(i => i.TargetMethod.ContainingType?.ToDisplayString()
        .StartsWithIgnoreCase(typePrefix) == true);

// Matching against an invocation operation
foreach (var (_, matcher, method, isAsync) in Matchers)
{
    if (matcher.Matches(invocation))
    {
        // matched — extract call site data
    }
}
```

**Pattern:** Build a static matcher table, then iterate + match in the analyzer. This keeps
detection declarative and the analysis loop clean.

#### Symbol Extensions

```csharp
// Attribute detection (TracedCallSiteAnalyzer)
if (method.HasAttribute(noTraceAttributeType))
    return null;

// Symbol equality (TracedCallSiteAnalyzer)
var tracedAttr = method.GetAttributes()
    .FirstOrDefault(a => a.AttributeClass.IsEqualTo(tracedAttributeType));

// Inheritance check (DbCallSiteAnalyzer)
if (!containingType.IsOrInheritsFrom(dbCommandType))
    return null;
```

#### AttributeExtensions — Extract attribute values without manual plumbing

**qyl analyzers currently do manual `ConstructorArguments[0].Value` access. Use these instead.**

```csharp
// BEFORE (manual, verbose, fragile)
var attr = method.GetAttributes()
    .FirstOrDefault(a => a.AttributeClass.IsEqualTo(tracedAttributeType));
var name = (string?)attr?.ConstructorArguments[0].Value;
var kind = attr?.NamedArguments
    .FirstOrDefault(na => na.Key == "Kind").Value.Value as string;

// AFTER (single call, null-safe, type-checked)

// On AttributeData directly:
var name = attr.GetConstructorArgument<string>(0);
var kind = attr.GetNamedArgument<string>("Kind");

// On ISymbol (finds attribute + extracts in one call):
var name = method.GetAttributeConstructorArgument<string>(
    "Qyl.Instrumentation.TracedAttribute", 0);
var kind = method.GetAttributeNamedArgument<string>(
    "Qyl.Instrumentation.TracedAttribute", "Kind");

// Try-pattern for conditional extraction:
if (method.TryGetAttributeConstructorArgument<string>(
    "Qyl.Instrumentation.TracedAttribute", 0, out var activitySource))
{
    // activitySource is guaranteed non-null here
}

// Array arguments (e.g. [MyAttr(new[] { "a", "b" })])
var values = attr.GetConstructorArgumentArray<string>(0); // ImmutableArray<string>

// Named argument arrays
var tags = attr.GetNamedArgumentArray<string>("Tags");

// Type arguments (e.g. [JsonDerivedType(typeof(Derived))])
var derivedType = attr.GetConstructorArgument<INamedTypeSymbol>(0);
```

**Full API surface:**

| Method | On | Returns |
|--------|----|---------|
| `GetConstructorArgument<T>(index)` | `AttributeData` | `T?` |
| `TryGetConstructorArgument<T>(index, out T)` | `AttributeData` | `bool` |
| `GetConstructorArgumentCount()` | `AttributeData` | `int` |
| `GetConstructorArgumentArray<T>(index)` | `AttributeData` | `ImmutableArray<T>` |
| `GetNamedArgument<T>(name)` | `AttributeData` | `T?` |
| `TryGetNamedArgument<T>(name, out T)` | `AttributeData` | `bool` |
| `HasNamedArgument(name)` | `AttributeData` | `bool` |
| `GetNamedArgumentNames()` | `AttributeData` | `IEnumerable<string>` |
| `GetNamedArgumentArray<T>(name)` | `AttributeData` | `ImmutableArray<T>` |
| `GetAttributeConstructorArgument<T>(attrName, index)` | `ISymbol` | `T?` |
| `GetAttributeConstructorArgument<T>(attrType, index)` | `ISymbol` | `T?` |
| `TryGetAttributeConstructorArgument<T>(attrName, index, out T)` | `ISymbol` | `bool` |
| `GetAttributeNamedArgument<T>(attrName, argName)` | `ISymbol` | `T?` |
| `GetAttributeNamedArgument<T>(attrType, argName)` | `ISymbol` | `T?` |
| `TryGetAttributeNamedArgument<T>(attrName, argName, out T)` | `ISymbol` | `bool` |
| `GetDisplayName(displayAttrType?)` | `ISymbol` | `string` |

### Anti-patterns — What NOT to do

```csharp
// DON'T use raw arrays in pipeline models
record Bad(string[] Items);                // breaks incremental caching
record Good(EquatableArray<string> Items); // correct

// DON'T use .Collect() when .CollectAsEquatableArray() exists
var bad = pipeline.Collect();                  // ImmutableArray, no value equality
var good = pipeline.CollectAsEquatableArray(); // EquatableArray, caching works

// DON'T do manual symbol comparison
if (symbol.Name == "IDisposable")                          // fragile string match
if (SymbolEqualityComparer.Default.Equals(a, b))           // verbose
symbol.IsEqualTo(other)                                    // use this instead

// DON'T manually dig into ConstructorArguments[i].Value
var bad = (string?)attr.ConstructorArguments[0].Value;     // no bounds check, no type safety
var good = attr.GetConstructorArgument<string>(0);         // null-safe, bounds-checked

// DON'T use raw StringBuilder for generated code
var bad = new StringBuilder();
bad.AppendLine("    {");                                   // manual indent tracking
var good = new IndentedStringBuilder();
using (good.BeginBlock()) { }                              // automatic indent + close

// DON'T use the *Context classes (OTelContext, AspNetContext, etc.)
// qyl uses InvocationMatcher/Invoke DSL for call-site detection instead.
// The Context classes are for ANcpLua.Analyzers, not qyl generators.

// DON'T duplicate utilities that exist in the package
// ErrorFingerprinter/ErrorCategorizer are duplicated between collector and loom.
// Use ProjectReference instead of copying.
```
