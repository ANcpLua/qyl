# Package Update API Matrix

Checked on 2026-06-16 from the Rider package-update screenshot, NuGet stable metadata, NuGet flat-container `.nuspec` source commits, and GitHub release tag/source diffs.

Scope notes:

- CI, workflow, dependency-bot, generated lockfile-only, and build-infrastructure-only changes were ignored when summarizing API surface.
- NuGet evidence uses the target package version page. GitHub evidence uses release tags when available, otherwise the source commit delta recorded in the NuGet `.nuspec`.
- Public API deltas were checked against NuGet metadata, package changelogs, GitHub release/tag diffs, and assembly inspection with `ilspycmd` where the package exposes a relevant assembly.
- `Microsoft.Agents.AI.Hosting` was not visible as a stable screenshot row because it is prerelease-only, but it was updated with the MAF family because `1.10.0-preview.260610.1` is the matching package that depends on `Microsoft.Agents.AI` `1.10.0`.

## API surface highlights

- `OpenAI` `2.11.0` adds `OpenAI.Skills.SkillClient`, skill and skill-version CRUD APIs, chat web-search context sizing, Responses token log-probability types/properties, streaming function-name events, and extensibility `Kind` APIs.
- `OpenTelemetry.Exporter.OpenTelemetryProtocol` `1.16.0` adds `OtlpExportCompression` and `OtlpExporterOptions.Compression`; the package changelog also records `OTEL_EXPORTER_OTLP_COMPRESSION=gzip` support.
- `Microsoft.Extensions.*` `10.7.0` promotes `ResourceQuota` / `ResourceQuotaProvider` to stable, promotes `HostedFileContent.SizeInBytes` and `HostedFileContent.CreatedAt` to stable, and fixes the OpenAI adapter so object-valued JSON Schema `additionalProperties` survives conversion.
- `Google.Protobuf` `3.35.1` adds `MessageExtensions.WriteLengthPrefixedTo(IMessage, IBufferWriter<byte>)` plus C# feature reflection types.
- `Grpc.AspNetCore` `2.80.0` adds `GrpcServiceEndpointConventionBuilder.Finally`, v1 reflection service support, and a trailers-only streaming fix.
- `Grpc.Tools` `2.81.1` adds a C# code-generation option to append `Async` to server-side method names.
- `DuckDB.NET.Data.Full` `1.5.3` improves table-function and parameter binding surface, including `CardinalityHint`, `TableFunctionInitData`, `ProjectedColumn`, and `ICollection` parameter binding.
- `Microsoft.Agents.AI` `1.10.0` adds `NonApprovalRequiredFunctionBypassingChatClient` and `ToolApprovalAgentOptions`.
- `Qyl.Api.Contracts` `0.2.0` adds telemetry-control-graph DTOs.
- Many `.NET` `10.0.9` and security-override packages had no new public API beyond servicing or patched transitive dependency coverage.
