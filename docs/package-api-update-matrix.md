## API surface highlights

- `OpenTelemetry.Exporter.OpenTelemetryProtocol` `1.16.0` adds `OtlpExportCompression` and `OtlpExporterOptions.Compression`; the package changelog also records `OTEL_EXPORTER_OTLP_COMPRESSION=gzip` support.
- `Microsoft.Extensions.*` `10.7.0` promotes `ResourceQuota` / `ResourceQuotaProvider` to stable and promotes `HostedFileContent.SizeInBytes` and `HostedFileContent.CreatedAt` to stable. (The OpenAI-adapter `additionalProperties` fix shipped in `Microsoft.Extensions.AI.OpenAI` `10.7.0`, which qyl no longer references — see that row.)
- `Google.Protobuf` `3.35.1` adds `MessageExtensions.WriteLengthPrefixedTo(IMessage, IBufferWriter<byte>)` plus C# feature reflection types.
- `Grpc.AspNetCore` `2.80.0` adds `GrpcServiceEndpointConventionBuilder.Finally`, v1 reflection service support, and a trailers-only streaming fix.
- `Grpc.Tools` `2.81.1` adds a C# code-generation option to append `Async` to server-side method names.
- `DuckDB.NET.Data.Full` `1.5.3` improves table-function and parameter binding surface, including `CardinalityHint`, `TableFunctionInitData`, `ProjectedColumn`, and `ICollection` parameter binding.
- `Microsoft.Agents.AI` `1.11.0` adds 13 public types — the stable net8.0+ evaluation records `GeneratedEvaluatorRef` and `RubricScore`, and the `[Experimental]` "Loop agent" family (`LoopAgent` plus `LoopEvaluator` strategies: AI-judge, completion-marker, todo-completion, delegate) — plus new members on `AgentEvaluationResults`, `FileAccessProvider`, and `AgentFileStore`.
- `ANcpLua.Agents` `1.11.0` is a telemetry **shrink** toward MAF 1.11's native `OpenTelemetryAgent`: it adds `UseAgentTelemetry` but removes the `AgentTelemetryNames` / `AgentTelemetryOptions` types and the `UseAgentRunTelemetry` / `UseAgentToolTelemetry` methods (breaking).
- `ANcpLua.Roslyn.Utilities` `2.2.36` adds the `Delta` / `SetDelta<T>` set-difference API for incremental-generator diffing and the `TypeDeclarationInfo` family (`ContainingTypeInfo`, `TypeDeclarationScope`) for injective hint names and nested partial-type scaffolding, plus `SyntaxTree`-bound `ToLocation` / `ToDiagnostic` overloads on `LocationInfo` / `DiagnosticInfo`.
- `Qyl.Api.Contracts` is now at `0.4.0` — a net contract **shrink**: `0.3.0`/`0.4.0` strip the 1.41-era deprecated-semconv migration fields (`gen_ai.*`, legacy `http.*` / `db.*` keys), repoint the OpenAI vendor wire names to their `openai.*` successors, and delete the `OTelVersion` / `HttpVersions` / `DbVersions` enums; the telemetry-control-graph DTOs added in `0.2.0` are unchanged.
- `Qyl.OpenTelemetry.SemanticConventions` family `3.4.0` — `3.2.0` re-pins the schema URL to semconv `1.43.0` and adopts the 1.42 GenAI two-repo split in the Incubating assembly; `3.3.0` is the full 1.43.0-surface regen (stable grows 185 -> 336 consts, but **removes** 37 `DbAttributes.SystemNameValues` members and `RequestMethodValues.Query`; Incubating drops the deprecated `gen_ai.system` / `gen_ai.usage.prompt_tokens` / `.completion_tokens` consts and the GenAI `SystemValues` class); `3.4.0` is analyzers/docs only, with zero runtime API delta. qyl is insulated from the breaking removals: the collector consumes the generated `CollectorSemanticAttributeCatalog`, never the SemConv types directly (enforced by `VerifyCollectorUsesSemanticConstants`).
- `Qyl.OpenTelemetry.AutoInstrumentation` `4.0.3` (first tracked at `3.1.2`): the injection library. `4.0.0` removed the public `QylInterceptedAspNetCore.Build(WebApplicationBuilder)` interceptor and added `AddQylAspNetCoreInstrumentation()` (server-span middleware via `IStartupFilter`); `4.0.1`–`4.0.3` gave DiagnosticListener spans real duration, added `url.scheme` / `http.request.method_original` semconv (+ the `UrlScheme` / `CodeFunctionName` consts), and introduced an internal single-owner-per-signal registry so the interceptor / middleware / DiagnosticListener lanes emit exactly one span per operation. Exposes `QylAutoInstrumentationOptions`, `QylActivitySource` / `QylActivityNames` / `QylAutoInstrumentationIds`, and a `.Hosting` `AddQylAutoInstrumentation(...)` entrypoint.
- Many `.NET` `10.0.9` and security-override packages had no new public API beyond servicing or patched transitive dependency coverage.
